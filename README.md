# Ideal

Armazenamento de arquivo para armazenar e recuperar dados de qualquer local

## Como funciona

- [ ] Cadastro e login

- [ ] Painel de controle

- [ ] Registro de domínios

- [ ] Gestão de projetos

- [ ] Gestão de arquivos

- [ ] Duvidas frequentes

## Como começar a usar

Instale o [Docker](https://www.docker.com/)

Baixe o projeto

```
bash env/pull.sh
```

Execute o projeto

```
bash env/up.sh
```
