# Ideal

Armazenamento de arquivo para armazenar e recuperar dados de qualquer local

## Como funciona

- [ ] Cadastro e login

- [ ] Painel de controle

- [ ] Registro de domínios

- [ ] Gestão de projetos

- [ ] Gestão de arquivos

- [ ] Duvidas frequentes

## Como começar a usar

Com o [Docker](https://www.docker.com/) instalado no Linux, execute:

```
bash env/up.sh
```
